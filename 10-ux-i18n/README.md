# UX Writing & i18n 로컬라이제이션 시스템

> 250+ UI 메시지 개선 + Lokalise 기반 10,000+ 문자열 다국어 자동화
> 프로젝트: 키퍼 서비스 어드민/앱 전체

---

## 프로젝트 개요

키퍼 서비스의 **UX 라이팅 표준 수립**과 **다국어 로컬라이제이션 자동화**를 진행한 프로젝트.
158개 성공 알림 삭제, 50+ 에러 메시지 개선, 104개 용어 표준화를 통해 인지 부하를 줄이고, Lokalise + CLI 자동화로 10,000+ 문자열의 다국어 관리 체계를 구축했습니다.

## UX Writing 개선

### 개선 원칙

| 원칙 | 설명 |
|------|------|
| **인지 부하 최소화** | 사용자가 읽어야 할 텍스트 최소화 |
| **흐름 보호** | 불필요한 알림으로 사용자 집중 방해 금지 |
| **일관성** | 같은 상황 = 같은 메시지 패턴 |
| **용어 통일** | 업무→일감, 공간그룹→지점, 할일→체크리스트 |

### Before / After 주요 사례

#### 성공 스낵바 → 삭제 (Silent Success)

| Before | After |
|--------|-------|
| `{name} 워크스페이스가 생성되었습니다.` | (삭제) |
| `키퍼가 배정되었습니다.` | (삭제) |
| `수정 내용이 저장되었습니다.` | (삭제) |

> 성공은 UI 상태 변화(버튼 비활성화, 모달 닫힘, 데이터 갱신)로 충분히 전달됨. **158개 성공 알림 전체 삭제.**

#### 에러 메시지 → 명사형 간결화

| Before | After |
|--------|-------|
| `우선 순위를 긴급으로 변경하는데 실패했습니다.` | `우선 수행 요청 실패` |
| `수정 내용 저장에 실패했습니다. 다시 시도해 주세요.` | `저장 실패` |

> "다시 시도해 주세요"는 자명한 행동이므로 제거. 마침표 제거, 명사형 종결.

#### 유효성 검증 → 단순화

| Before | After |
|--------|-------|
| `초성, 중성, 종성은 단독으로 사용할 수 없습니다.` | `자음, 모음만 입력할 수 없습니다` |
| `할일 이름을 입력해주세요` | `이름을 입력해 주세요` |

> 기술 용어 → 일상어 전환. 필드명 중복 제거.

#### Empty 상태 → 핵심만

| Before | After |
|--------|-------|
| `공간 그룹이 아직 설정되지 않았습니다. 우측 상단의 '공간 그룹 추가' 버튼을 눌러 시작해 보세요.` | `등록된 지점이 없습니다` |

> 버튼 위치 안내, 과도한 친절함 제거. 패턴: `등록된 {항목}이 없습니다`

#### 삭제 확인 → 의문문에서 서술문으로

| Before | After |
|--------|-------|
| `정말로 이 할일 그룹을 삭제하시겠습니까?` | `체크리스트 그룹을 삭제합니다` |

> 사용자가 이미 삭제 버튼을 눌렀으므로 의도 확인 불필요. "정말" 제거, 서술문 전환.

#### 버튼 텍스트 표준화

| Before | After |
|--------|-------|
| 취소 | 닫기 |
| 계속진행 | 확인 |
| 추가 | 등록 |
| 선택 / 생성 | 저장 |

## i18n 로컬라이제이션 시스템

### 기술 스택

| 영역 | 기술 |
|------|------|
| **Platform** | Lokalise (Cloud-based L10n SaaS) |
| **CLI** | Lokalise CLI (파일 업로드/다운로드 자동화) |
| **Automation** | Python (Excel → JSON 변환 스크립트) |
| **Data Format** | JSON (Lokalise 호환 포맷) |

### 키 생성 규칙 (계층 구조)

```
메뉴.영역.대상.타입
```

| 계층 | 역할 | 예시 |
|------|------|------|
| **메뉴** | LNB 메뉴명 | 실시간, 공통, 배정 |
| **영역** | UI 섹션 | 헤더, 필터, 버튼, 목록 |
| **대상** | 치환 대상 | 날짜, 공간그룹, 검색 |
| **타입** | 기능 구분 | 제목, 라벨, 플레이스홀더, 텍스트 |

**적용 예시:**

| UI 요소 | 키 |
|---------|-----|
| 실시간 업무 현황 타이틀 | `실시간.헤더.제목` |
| 날짜 선택 라벨 | `실시간.필터.날짜.라벨` |
| 지점 검색 플레이스홀더 | `실시간.필터.공간그룹.플레이스홀더` |
| 저장 버튼 (공통) | `공통.버튼.저장.텍스트` |

### 자동화 워크플로우

```
Excel (용어/번역 관리)
    ↓ Python 변환 스크립트
JSON (Lokalise 포맷)
    ↓ Lokalise CLI upload
Lokalise (웹 번역 관리)
    ↓ Lokalise CLI download
JSON (앱/어드민 코드에 적용)
```

## 성과 및 효과

### UX 품질 향상
- **인지 부하 감소:** 158개 성공 알림 삭제로 사용자 흐름 보호 (Silent Success 패턴)
- **메시지 일관성:** 원칙 기반 톤앤매너 가이드로 250+ 메시지의 어조/형식 통일
- **용어 통일:** 104개 서비스 용어 표준화로 사용자 혼란 제거

### 운영 효율화
- **비개발자 참여:** Excel → JSON 자동 변환으로 기획자/디자이너가 직접 번역 관리 가능
- **CLI 자동화:** Lokalise CLI로 파일 업로드/다운로드 자동화 → 수동 작업 제거
- **계층적 키 체계:** 메뉴.영역.대상.타입 구조로 10,000+ 문자열을 체계적으로 관리

### 확장성
- **다국어 대응:** Lokalise 플랫폼 기반으로 신규 언어 추가 시 기존 워크플로우 그대로 활용
- **태그 기반 릴리즈:** 날짜 태그(20260212)로 릴리즈별 문자열 변경 추적 가능
